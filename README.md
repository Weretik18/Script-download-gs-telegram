# Script-download-gs-telegram
Telegram Voice Messages Downloader — консольный инструмент на Python для скачивания голосовых сообщений из Telegram-каналов. Поддерживает поиск каналов по ID, username или из списка подписок, автоматически сохраняет голосовые сообщения в формате OGG и создает файлы с метаданными.
